KeyMaster是一款自定义黑莓快捷键的软件，能够帮助扩展黑莓的快捷键，可以使用在BlackBerry ROM 4.3以上的机型。

使用方法：
第一步：把黑莓快捷键(Convenience Key)定义成KeyMaster，选项->屏幕/键盘->快捷键打开，选择KeyMaster
第二步：在KeyMaster里定义你的程序快捷键，菜单->添加程序,选择你要添加的程序，然后按a-z中任何一个键，就会自动把这个程序和对应快捷键添加进去。
第三步：这样做完后，只需2个按键就能启动任何手机内的程序，比如我添加了记事本程序[a],那么我按下Convenience Key，然后再按a就可以进入记事本程序。（注意：不是同时按，是先按Convenience Key再按a）
